MedRemind

MedRemind is a mobile application that provides timely medication reminders to users.
It is built using Flutter and Dart, making it a cross-platform solution for Android and iOS.
The app utilizes Supabase as its database, leveraging PostgreSQL for data management.

Key Features:

  • Customizable Medication Schedules
  • Timely Reminders and Alerts
  • Medication Dosage Tracking
  • Medication Image Upload
  • Inventory Overview
  • Secure Login with Google, Facebook, and GitHub
  • Intuitive User Interface

Technologies Used:

  • Flutter and Dart
  • Supabase (PostgreSQL)

How It Works:

When you open the app for the first time, you'll need to create an account.
This can be done manually with your email address or you can quickly log in with Google, Facebook, or GitHub.

Once logged in, you can set up your profile and start using the app.
Add new medication and attach an image to easily identify each pill.

After setting up your inventory, you can start creating reminders.
Choose specific days and times using the fancy time picker, or select a date range for medications that need to be taken over a period.
Need a one-time reminder? No problem. We have a button for that.

The overview page shows the medication you need to take today.
Confirm the medication at the overview page (confirmation resets after midnight if the reminder repeats).
Can't take the medication immediately? No worries, you can add a 30-minute delay to receive a reminder later.

User Interface:

The app's user interface is designed with a focus on user-friendliness, catering to people of various age groups.
Extensive testing was conducted, involving individuals between the ages of 20 and 75, to ensure that everyone can comfortably navigate and utilize the application.

To address the needs of older users who may have multiple medications to manage, we implemented a thoughtful feature.
Users can easily add images of their medications, whether from their gallery or taken at the moment.
This visual aid assists in identifying the correct medication, reducing any confusion.

Additionally, we included a helpful threshold notification system. When medication is running low, the app will provide a warning, prompting users to take timely action.

The main color scheme adopts a medical blue/grey theme, making it distinctively recognizable as a medical app.
Upon opening the app, users are greeted with an intuitive home screen that displays a comprehensive overview of the medications they need to take on that day.
Quick accessibility is ensured through a bottom navigation bar, granting swift access to essential features.
Furthermore, the hamburger menu provides easy access to the settings page, logout functionality, and leaves room for future expansions.

Database Management:

MedRemind utilizes Supabase, a versatile platform that facilitates seamless integration of database, image storage (storage bucket), and social media login functionalities.
With Supabase's database feature, we efficiently manage and access user data using standard SQL queries, ensuring a smooth user experience.

To enhance user interaction, MedRemind leverages Supabase's image storage feature to securely store and retrieve medication images.
This allows users to add custom images for their medications, making it easier to identify and take the right pills at the right time.
The images are stored under the users map with the users UID, using row-level security (RLS) to ensure that only the right users can access the images.

Additionally, Supabase's social media login integration empowers MedRemind users to quickly log in using their existing Google, Facebook, or GitHub accounts.
This eliminates the need for creating new credentials, simplifying the onboarding process and enhancing user engagement.
